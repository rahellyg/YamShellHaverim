# YamShellHaverim

אפליקציית ווב קלה להפעלת קהילת חברים מהכיתה:

- מציאת חברים מכיתה א׳ לפי מרחק קרוב
- ניהול מפגשים עם תאריכים (במצב מנהל)
- ארכיון תמונות
- פעילויות התנדבות ותרומות להפקת אירועים

## הרצה מקומית

1. לפתוח טרמינל בתיקיית הפרויקט.
2. להריץ:

```bash
python3 -m http.server 8000
```

3. לפתוח בדפדפן:

```text
http://localhost:8000
```
