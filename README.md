# YamShellHaverim

אפליקציית ווב קלה להפעלת קהילת חברים מהכיתה:

- **מערכת אימות מלאה** - כניסה עם אימייל/סיסמה או Google
- מציאת חברים מכיתה א׳ לפי מרחק קרוב
- ניהול מפגשים עם תאריכים (במצב מנהל בלבד)
- ארכיון תמונות
- פעילויות התנדבות ותרומות להפקת אירועים

## מערכת אימות

### משתמש רגיל
- הירשם עם אימייל וסיסמה
- או התחבר עם Google
- גישה לכל התכונות למעט ניהול מפגשים

### מנהל
**פרטי כניסה מוגדרים מראש:**
- אימייל: `admin@yamshell.com`
- סיסמה: `Manager2026!`
- גישה מלאה לכל התכונות כולל ניהול מפגשים

📖 **למדריך מלא לאימות:** ראה [AUTH_SETUP.md](AUTH_SETUP.md)

## הרצה מקומית

1. לפתוח טרמינל בתיקיית הפרויקט.
2. להריץ:

```bash
python3 -m http.server 8000
```

3. לפתוח בדפדפן:

```text
http://localhost:8000
```
